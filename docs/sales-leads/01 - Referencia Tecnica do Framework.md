# Referência Técnica do Framework de Comercialização

**Versão:** 1.0  
**Uso:** Material de referência para a skill sales-leads

---

## As 4 Sub-fases

```
Qualificação → Educação → Oferta → Fechamento
```

---

## 1. Qualificação

### Os 3 métodos

| Tipo | Mecanismo | Precisão | Escala |
|---|---|---|---|
| **Autoprovação** | Landing page com critérios, quiz, self-assessment | Média | Alta |
| **Comportamental** | Scoring por ações (abriu email, visitou preço) | Alta | Alta |
| **Humana** | SDR, telefonema, reunião de diagnóstico | Muito alta | Baixa |

### Quando usar

| Volume de leads | Ticket médio | Melhor abordagem |
|---|---|---|
| Alto (>1000/mês) | Baixo (<$100) | Autoprovação |
| Médio (100-1000/mês) | Médio ($100-$1000) | Autoprovação + Comportamental |
| Baixo (<100/mês) | Alto (>$1000) | Humana |

---

## 2. Educação

### Os 3 métodos

| Abordagem | Mecanismo | Ticket ideal |
|---|---|---|
| **Sequência automatizada** | Email/MSG série (identificação → agravação → solução → prova → oferta → urgência) | Baixo/Médio |
| **Conteúdo sob demanda** | Demo, webinar, case study, tutorial, whitepaper | Médio |
| **Conversa direta** | Reunião, call, chat especialista | Alto |

### Diferenças por tipo de produto

| | Digital/SaaS | Físico |
|---|---|---|
| Maior barreira | "Vai funcionar pra mim?" | "Vale o preço?" |
| Melhor prova | Demo gratuita, trial | Amostra, teste físico |
| Conteúdo mais eficaz | Tutorial + case | Catálogo + review + unboxing |
| Decisão | Racional + behavioral | Sensorial + emocional + racional |

---

## 3. Oferta

### Os 3 tipos

| Tipo | Mecanismo | Complexidade |
|---|---|---|
| **Direta** | "Compre agora por X" | Baixa |
| **Condicionada** | Desconto limitado, bundle, condição especial | Média |
| **Proposta** | Personalizada, negociação, contrato | Alta |

### Os 4 elementos de uma oferta completa

1. **O que** — Produto/serviço + bônus + complementos
2. **Quanto** — Preço + condições de pagamento
3. **Risco** — Garantia, devolução, trial
4. **Urgência** — Por que agora

### Gatilhos de urgência

- Tempo — "Preço promocional só até sexta"
- Quantidade — "Restam 20 vagas"
- Condição — "Primeiros 100 clientes ganham bônus X"
- Bundle — "Compre A e receba B grátis"

---

## 4. Fechamento

### Os 3 tipos

| Tipo | Mecanismo | Ticket ideal |
|---|---|---|
| **Automático** | Checkout online, 1 clique | Baixo |
| **Semi-automático** | Chatbot, countdown, retargeting | Médio |
| **Humano** | Vendedor conduz negociação | Alto |

### Táticas semi-automáticas

- Abandono de carrinho — email/SMS em 1h, 24h, 72h
- Chatbot de objeções
- Countdown/urgência no checkout
- Retargeting para visitantes da página de preço
- Prova social em tempo real

### Estrutura de fechamento humano

```
1. Recap — "Pelo que entendi, você precisa de X e Y"
2. Confirmação — "Isso mesmo? Falta algo?"
3. Apresentação final — "Aqui está a proposta completa"
4. Tratamento de objeções — "Entendo. O que fazemos é..."
5. Assumptive close — "Quando gostaria de começar?"
```

### Barreiras do fechamento

1. **Objeções** — preço, timing, concorrência
2. **Atrito logístico** — checkout confuso, pagamento limitado
3. **Hesitação emocional** — medo de errar, medo de arrependimento

---

## 5. Matriz de Decisão por Tipo de Produto

| Característica | Digital/SaaS | Físico | Serviço | Mídia/Conteúdo |
|---|---|---|---|---|
| **Qualificação ideal** | Comportamental | Autoprovação | Humana | Comportamental |
| **Educação ideal** | Sequência + Demo | Sequência | Conversa direta | Conteúdo sob demanda |
| **Oferta ideal** | Direta/Condicionada | Direta | Proposta | Condicionada |
| **Fechamento ideal** | Automático | Semi-automático | Humano | Automático |
| **Maior objeção** | "Vai funcionar?" | "Vale o preço?" | "Posso confiar?" | "Vai ter retorno?" |
| **Melhor resposta** | Trial + garantia | Amostra + devolução | Portfólio + referência | Cases + métricas |
| **Ciclo de fechamento** | Minutos a dias | Dias a semanas | Semanas a meses | Dias a semanas |
